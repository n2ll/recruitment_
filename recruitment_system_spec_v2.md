## 1. 시스템 개요

배달/운송 기사 채용 전 과정을 자동화하는 시스템.
모집 → 서류심사 → 전화스크리닝 → 확정/대기 → 온보딩 → 현장투입까지의 파이프라인을 구축한다.

### 핵심 원칙
- **셀레니움 사용 절대 금지** (외부 플랫폼 자동화 X)
- **Supabase가 메인 DB**, 구글 시트는 담당자용 대시보드
- **외부 노코드 도구(Make 등) 미사용** — Next.js API route에서 직접 처리
- 메시징은 **SOLAPI 문자 발송** (등록 완료)
- 광고는 **메타 광고** (준비 완료)

## 5. 데이터 저장

### 구조

```
지원자 폼 제출
    ↓
Supabase applicants 테이블 저장 (메인 DB)
    ↓
구글 시트에 동기화 (담당자용 대시보드)
```

- **Supabase**: API, 중복 체크, 데이터 안정성
- **구글 시트**: 담당자가 보고 + 상태 변경하는 용도

### Supabase 테이블 (applicants)

`supabase-schema.sql` 참고

### 구글 시트 역할
- 담당자가 지원자 목록 확인 (필터 뷰로 본인 지점만)
- 진행상황 셀 변경 → 앱스크립트가 감지 → API 호출 → 문자 발송
- 슬롯 현황 시트에서 잔여 슬롯 확인

## 6. 자동화 흐름

### 전체 플로우

```
모집 채널 (메타/카카오/알바몬)
          ↓
커스텀 폼 (Next.js)
?source=meta&branch=광진자양
          ↓
Supabase 저장 + 구글 시트 동기화 + 슬랙 알림
          ↓
담당자 전화 스크리닝
(관리자 대시보드에서 확인)
          ↓
대시보드에서 스크리닝 완료 클릭
  ↙              ↘
"온보딩"          "부적합" / "대기"
  ↓                ↓
SOLAPI 문자 자동   부적합/대기 안내 문자
발송 (배민커넥트)   자동 발송
  ↓
지원자 문자 회신 (전용 폰 수신)
  ↓
SMS Gateway → Supabase messages 테이블
  ↓
관리자 대시보드 대화 화면에서 확인 + 답장
  ↓
진행상황 = "현장투입"
```

### 문자 송수신 흐름

```
[발신 (outbound)]
관리자 대시보드 → 솔라피 API → 지원자 수신
                  ↓
              messages 테이블 저장
              (direction: outbound, sent_by: 관리자)

[수신 (inbound)]
지원자 문자 발송 → 전용 폰 수신 → SMS Gateway 앱
                                    ↓
                                Room DB 저장
                                    ↓
                                Supabase messages POST
                                (direction: inbound)
                                    ↓
                                트리거: applicants.unread_count +1
                                    ↓
                                관리자 대시보드 폴링으로 확인
```

### 트리거 목록

| 트리거 | 실행 | 구현 방식 |
|--------|------|-----------|
| 폼 제출 | Supabase 저장 + 구글 시트 추가 + 슬랙 알림 | API route |
| 스크리닝 완료 클릭 | 배민커넥트 가입 안내 문자 발송 | 대시보드 → API |
| 대시보드에서 문자 발송 | 솔라피 발송 + messages 저장 | API route |
| 전용 폰 SMS 수신 | messages 테이블 저장 + unread_count 증가 | SMS Gateway + DB 트리거 |
| 전화번호 중복 | 자동 감지 + 비고에 표시 | API route |
| 전용 폰 heartbeat | 5분마다 상태 ping | SMS Gateway WorkManager |
